# Overview

This repository is designed to organize and document the core concepts of web development.

It includes structured notes and examples for:

* HTML fundamentals
* CSS layout and styling
* JavaScript concepts
* React for frontend development
* Node.js and Express for backend development
* MongoDB for database management

The goal is to maintain **clear and concise notes** for revision and interview preparation.

# Frontend Topics

Frontend development focuses on building the user interface and user experience of web applications.

Technologies covered:

* HTML
* CSS
* JavaScript
* React

Key areas include:

* Page structure and semantic HTML
* Responsive layouts
* DOM manipulation
* Component-based UI development

---

# Backend Topics

Backend development focuses on server-side logic, APIs, and database interactions.

Technologies covered:

* Node.js
* Express.js
* MongoDB

Key areas include:

* REST API development
* Server architecture
* Database integration
* Authentication and middleware
